# Airbnb Clone: Clone the Airbnb Mobile App using React Native                     
1. Connected a backend with Authentication and GraphQL using AWS Amplify. 
2. Integrated Google Places Autocomplete and Maps API to present the homes interactively.
3. Designed the UI components with hooks and navigated them organically with React Navigation.
